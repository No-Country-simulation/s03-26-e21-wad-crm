## [1.0.0] - 2026-03-17

### 🎉 Versión inicial

#### Agregado
- **Epic**: Startup CRM - Prototipo funcional
- **Módulos**:
  - 🔐 Autenticación - Login, registro, recuperación de contraseña
  - 📇 Contactos - CRUD, empresas, segmentación
  - 💬 WhatsApp - Integración WhatsApp Cloud API
  - 📧 Email - Integración SMTP/Brevo
  - 📊 Métricas - KPIs y visualización
  - ⚙️ Configuración - Cuenta, automatizaciones

#### Herramientas integradas
- Next.js 14 con App Router
- TypeScript (modo estricto)
- Tailwind CSS + shadcn/ui
- Zustand para estado global
- MSW para mocking de APIs
- ESLint + Prettier
- Storybook
