# HU-020: Crear Empresa

**Módulo**: Contacts  
**Prioridad**: Media  
**Como**: vendedor  
**Quiero**: registrar una empresa  
**Para**: agrupar contactos de la misma organización

## Criterios de Aceptación

- [ ] Formulario con campos: nombre, website, industria, tamaño, dirección
- [ ] Empresa puede tener múltiples contactos
- [ ] Validación de nombre único dentro del workspace
- [ ] Al crear empresa, se puede agregar un contacto inicial
- [ ] Vista de empresa muestra lista de contactos relacionados
