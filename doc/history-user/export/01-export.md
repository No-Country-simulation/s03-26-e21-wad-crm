# HU-090: Exportar Contactos a CSV

**Módulo**: Export  
**Prioridad**: Alta  
**Como**: usuario  
**Quiero**: exportar mis contactos a CSV  
**Para**: usarlos en otras herramientas

## Criterios de Aceptación

- [ ] Botón "Exportar" en lista de contactos
- [ ] Selector de columnas a incluir
- [ ] Filtros actuales se aplican a la exportación
- [ ] Preview de cantidad de registros
- [ ] Descarga directa de archivo CSV
- [ ] Encoding UTF-8
